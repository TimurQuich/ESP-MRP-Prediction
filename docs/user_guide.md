# 📖 Руководство пользователя

## Введение

Приложение для прогнозирования межремонтного периода (МРП) электроцентробежных насосов (ЭЦН) с использованием машинного обучения.

---

## 🚀 Запуск

```bash
streamlit run app.py
```

После запуска откройте в браузере: `http://localhost:8501`

---

## 📊 Загрузка данных

### Способ 1: С GitHub (по умолчанию)
- Оставьте галочку **"Загрузить с GitHub"**
- Нажмите **"🚀 Запустить анализ"**

### Способ 2: Локальный файл
- Снимите галочку **"Загрузить с GitHub"**
- Нажмите **"Загрузите свой файл"**
- Выберите `Dataset_for_ESP.xlsx`
- Нажмите **"🚀 Запустить анализ"**

---

## 📁 Требования к файлу

| Колонка | Тип | Описание |
|---------|-----|----------|
| `Модель насоса` | текст | Тип установки |
| `Глубина спуска, м` | число | Глубина установки |
| `% в` | число | Обводненность |
| `МРП` | число | Целевая переменная |
| `Qж факт` | число | Дебит жидкости |
| `Qн факт` | число | Дебит нефти |
| `Причина остановки` | текст | Причина отказа |

> **Важно!** Названия колонок должны совпадать **точно**.

## 🎯 Вкладки

### 1. 📊 Данные
- Количество записей и признаков
- Первые 5 строк
- Статистика

### 2. 📈 Распределение МРП
- Гистограмма с кривой плотности
- Среднее, медиана, квартили

### 3. 🤖 Сравнение моделей
- 4 модели: Linear Regression, Random Forest, Gradient Boosting, Extra Trees
- Метрики: MAE, RMSE
- График "Факт vs Предсказание"

### 4. ⚡ Оптимизация
- GridSearchCV – полный перебор
- RandomizedSearchCV – случайный поиск
- Optuna – байесовская оптимизация

### 5. 📉 Анализ важности признаков
- Топ-10 важных признаков
- Корреляционная матрица

---

## 📊 Интерпретация

**Метрики:**
- MAE < 50 – отличная модель
- MAE 50–100 – хорошая модель
- MAE > 100 – требуется улучшение

**Корреляция:**
- Красный – положительная связь
- Синий – отрицательная связь
- МРП выделен жирным
